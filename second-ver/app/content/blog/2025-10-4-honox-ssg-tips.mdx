## よくあるハマり

- `import.meta.glob` は**パスがズレると一生見つからない**（パス確認！）
- 画像は `/public/**` に置こう（Vite の静的配信）

```ts title="app/routes/blog/index.tsx の注意"
const modules = import.meta.glob('/content/blog/**/*.mdx', { eager: true });
```
